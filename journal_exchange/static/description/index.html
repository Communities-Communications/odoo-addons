Exchange Currencies

Handle your Exchange Currencies between Journals

You can transfer between journals with different currencies.

System suggestions

Based on your currencies configuration

System suggest you how much money you have to receive.

You can fill Exchange Rate or Exchange Amount (how much did you really received)

Automatic

This operation creates a journal entry

System is going to create a journal entry based on data provided.

You don't have to be worried about the journal entry, it's created with currency losses or gains.

Configurable

This module uses native Accounting configuration for currencies.
